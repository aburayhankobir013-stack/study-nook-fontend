# 📚 Study Nook Frontend

Study Nook is a modern study room booking platform that helps students discover, reserve, and manage study spaces efficiently. The frontend provides a seamless and responsive user experience for browsing rooms, making bookings, and managing reservations.

---

## 🌟 Features

* 🔐 Secure Authentication
* 🏢 Browse Available Study Rooms
* 🔍 View Detailed Room Information
* 📅 Book Study Rooms
* 📝 Manage Personal Bookings
* ✏️ Update Booking Details
* ❌ Cancel Reservations
* 📱 Fully Responsive Design
* ⚡ Fast Navigation with Next.js
* 🎨 Modern and Clean User Interface

---

## 🚀 Technologies Used

* Next.js
* React.js
* Tailwind CSS

* Better Auth
* React Hook Form
* React Icons

## 🔐 Authentication

Study Nook uses Better Auth for secure authentication and session management.

Features include:

* User Registration
* User Login
* Session Persistence
* Route Protection
* Secure Logout
